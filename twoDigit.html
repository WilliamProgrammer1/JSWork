For this, it counts as a 2-digit number if it has two digits in a row (must be 0 to 9 for a digit) where the next or previous character is anything except another digit (alternatively, there could be no previous or next character).

When the user clicks the button, you should use an alert dialog to let them know the result. You can simply reply with Works and Nope or something similar...
