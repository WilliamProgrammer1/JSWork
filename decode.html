Description

Enter a message in binary or hex and this will decode it for you and show (via an alert dialog) the decoded message in plain English. Enter binary numbers one byte at a time; put spaces between the bytes as a separator (e.g. 01000010 01100101 011...). Enter hex numbers as two-digit hex values (each representing a separate symbol); put spaces between the hex values as a separator (e.g. 4E 4F 50 45).

NOTE: These values must come from the ASCII table ; for example, the binary value associated with A is 01000001 while the hex value associated with A is 41 (note that is HEX... the decimal value associated with A is 65).


A Helper Function

Take a look at the String's fromCodePoint() function (click here and then scroll down just a touch). If you send it a number (either a decimal, hex, or binary number), then it will give you back a symbol/letter. The fromCodePoint() function is expecting a string parameter...but formatted in a particular way.

Binary number format for JS: any number starting with 0b

Hex number format for JS: any number starting with 0x
